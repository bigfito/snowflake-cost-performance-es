# snowflake-cost-performance-es

Hola.  Este repositorio tiene la intención de compartir una serie de Notebooks de Snowflake que contienen consultas SQL de ayuda para analizar temas como costos por cómputo, almacenamiento y data transfer para una cuenta de Snowflake.  Así como también analizar información concerniente al desempeño de consultas SQL (jobs) en los virtual warehouse con que estemos trabajando.  Los notebooks han sido desarrollados tomando en cuenta la documentación oficial de Snowflake.

Para empezar a trabajar con estos notebooks, solo tienes que importar el notebook en SNOWFLAKE NOTEBOOKS y asociar un VIRTUAL WAREHOUSE para que puedas ejecutar las consultas.  Con un virtual warehouse de tamaño X-SMALL es más que suficiente.

La documentación oficial de Snowflake la puedes encontrar en: https://docs.snowflake.com.
