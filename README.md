# snowflake-cost-performance-es
Hola.  Este repositorio tiene la intención de compartir una serie de Notebooks de Snowflake que contienen consultas SQL de ayuda para analizar temas como costos por cómputo, almacenamiento y data transfer para una cuenta de Snowflake.  Así como también analizar información concerniente al desempeño de consultas en los virtual warehouse.
